第一维度 : `Height` 高度, 对应这张图片的 `nRow`行数

第二维度 : `Width` 宽度, 对应这张图片的`nCol` 列数

第三维度: `Value` BGR三通道的值. 

BGR 分别代表

B: `Blue`  蓝色

G: `Green` 绿色

R: `Red` 红色
